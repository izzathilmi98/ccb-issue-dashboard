Overview

This dashboard shows one live view of all open and closed machine issues across TEM, WDB, and SDB. Data is read straight from each machine's issue tracker Excel file — nothing is typed here.

The dashboard is read-only. Use it for a daily health check before opening the source trackers. It auto-refreshes every 30 seconds — no need to reload the page manually.

Use the menu on the left to jump to any section, or scroll this page.

  5. 5Resolution Rate — Closed ÷ Total, as a percentage. This is a lifetime ratio (all issues ever logged) — it is NOT the same as Weekly Closure Rate below. A high Resolution Rate just means most historical issues eventually got closed; it says nothing about this week's pace.

A blank Issue Status cell is also treated as TBA (issue logged but not yet moved past the initial state). Any status text that doesn't match this list falls back to Open so it stays visible rather than silently disappearing.

TEM

TEM's status comes straight from the Power Automate export flow, not this phase-based system — just Open or Closed, set directly by the flow's own completion tracking (bucket/progress fields in Planner). No TBA, Monitoring, or On-Hold concept for TEM today.

Export Excel
